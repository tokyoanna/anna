1. O que é o Node.js?
Node.js é um ambiente de execução que permite rodar código JavaScript fora do navegador.

2. O Node.js usa qual linguagem de programação?
Ele usa JavaScript.

3. O Node.js roda no navegador ou fora dele?
Ele roda fora do navegador, no servidor ou diretamente no computador.

4. Para que serve o Node.js?
Serve principalmente para criar aplicações back-end, como servidores, APIs, sistemas web, aplicações em tempo real (chat), entre outros.

5. Qual comando é usado para executar um arquivo Node.js no terminal?

node nomeDoArquivo.js


6. O que é o npm?
npm (Node Package Manager) é o gerenciador de pacotes do Node.js. Ele serve para instalar, atualizar e gerenciar bibliotecas e dependências.

7. O que significa a palavra módulo no Node.js?
Módulo é um arquivo ou conjunto de arquivos que contém código reutilizável (funções, variáveis, classes) que pode ser importado em outros arquivos.

8. O Node.js é mais usado no front-end ou no back-end?
É mais usado no back-end.

9. O que é o arquivo package.json?
É um arquivo que contém informações sobre o projeto (nome, versão, scripts) e lista as dependências instaladas.

10. O Node.js é gratuito ou pago?
O Node.js é gratuito e de código aberto (open source).
